13C.01.02 Working with colors and animations

// Link to the solution

About the solution

For this assignment I created a color scheme with 4 colors using HueMint. I used a dark purple and white as my primary colors with a strong pink color as an accent color to use for links and CTA's. I used a color contrast checker to validate the text's readability with light - and darkmode. To implement dark mode I used CSS variables and Local storage which I implemented through JavaScript. In terms of animations I added scroll-triggered animations for sections and a "progress bar" animation using MotionOne animation framework.
