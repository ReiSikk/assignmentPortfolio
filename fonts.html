About the solution

For this assignment I decided to use the font Poppins for the headers and Inter for body text. I used the Perfect Fourth scale for setting up the visual hierarchies in text use. I combined the use of different font sizes, font weights and lightness of the text in order to define the visual hierarchy. I used flex to layout the UI cards and a minimal amount of rounded borders for the UI cards and CTA's. Hover effects were added to CTA's to further enhance the experience. The UI cards have a clear visual hierarchy - the name of the person being the biggest in size and weight, the description with a different font, font size and weight and the CTA's with uppercase text that has extra letter spacing to make those stand out from the rest of the content.
