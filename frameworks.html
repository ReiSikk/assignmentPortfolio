About the solution

To solve this assignment I chose to use the CSS framework called Pico. Using Pico allowed me to quickly and conveniently make the website responsive using their pre-made classes for layouting. Along with the layouting classes Pico provides I used their buttons which made it easy to have consistently styled buttons. As the assignment was to design a responsive website I also added a burger menu navigation when the viewport scales down to a mobile size. The overall layout uses a classic column drop layout patter so that as the viewport gets smaller the grid columns stack on top of each other resulting in a single column view on the mobile.
